Altreurl

A compact request redirector for local backend debugging, with rules, credential sync, safe import/export, diagnostics, and contextual popup controls.

View extension repo Chrome Web Store copy

Built for backend developers

Switch app traffic to localhost without changing application code.

Altreurl v1.13.0 keeps the workflow close to the extension UI: searchable rules, grouped editor sections, import/export previews, privacy-safe diagnostics, and light/dark icon sets.

Core features

Redirects, credentials, and rule control in one extension.

URL

Wildcard and regex redirects

Match source requests with simple wildcards or precise regex capture groups, then redirect to localhost or another endpoint.

AUTH

Manual credentials

Set Authorization values and custom request headers by rule when a local backend needs explicit debugging context.

SYNC

Credential sync

Learn headers, Authorization values, and session cookies from source requests, browser storage, or Chrome cookies.

TAB

Contextual popup

See only the rules relevant to the active tab domain and enable or disable matching rules from the popup.

I/O

Mature import and export

Preview imported rules, handle conflicts, choose export scope, and redact or remove credentials before sharing JSON.

LOGS

Diagnostics and logs

Review privacy-safe extension state, dynamic rule counts, apply errors, and recent local events when debugging setup issues.

Example workflow

Point a remote API route at localhost.

Create a rule for the source API path, choose the localhost target, and decide whether credentials should be typed manually, learned from a request, read from storage, or read from cookies.

Source URL pattern https://api.example.com/users/* Redirect target URL http://localhost:3000/users/* Credential mode Sync from source Export safety Redact credentials before sharing

Privacy stance

No Altreurl server receives your rules or credentials.

Altreurl handles sensitive debugging data only for configured rules. It does not sell data, send browsing activity to an Altreurl server, or transfer rule exports unless you create the export file yourself.
